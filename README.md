# 📚 Library Management System

A simple **Library Management System** built using **HTML, CSS, and JavaScript**.
This project allows users to explore books, search by title, and filter books based on categories.

---

## 🚀 Features

* 📖 Display books dynamically using JavaScript
* 🔎 Search books by title
* 🗂 Filter books by category (Basic, IT, CSE, ISE, Novel, Fantasy)
* 🖼 Display book images with author and publication date
* 📱 Responsive user interface
* 🎨 Modern UI using Google Fonts and Font Awesome icons

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript (DOM Manipulation)** – Dynamic UI rendering

## ⚙ How It Works

1. Book information is stored in a **JavaScript data object**.
2. JavaScript dynamically creates book cards using **DOM manipulation**.
3. Users can:

   * Search for books by title
   * Filter books by category
4. The interface updates dynamically without reloading the page.
